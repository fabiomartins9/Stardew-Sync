🌾 Stardew Sync — Backup e Gerenciamento de Perfis

O Stardew Sync é um programa para Windows projetado para facilitar o gerenciamento, backup e duplicação de saves do jogo Stardew Valley.

Com ele, você pode criar cópias de segurança dos seus dados e duplicar seus perfis de jogo (fazendas) de forma rápida e organizada, permitindo testar diferentes estratégias, preservar versões antigas ou manter ambientes separados sem risco de perder progresso.

🎯 Objetivo

Este projeto surgiu de uma necessidade prática do dia a dia:

Eu jogo Stardew Valley e meu filho de 5 anos sempre quer jogar usando minha fazenda, já que tenho mais itens e progresso. Já fiz a cópia manualmente uma vez, mas percebi que isso se tornaria recorrente. Então decidi criar uma forma mais simples e rápida de fazer isso.

🚀 Principais funcionalidades
💾 Backup completo de saves
Crie cópias de segurança das suas fazendas com apenas um clique.
🧬 Duplicação de perfis (clonagem de saves)
Gere cópias independentes de uma fazenda para testar caminhos diferentes no jogo ou compartilhar com outras pessoas.
🔄 Sincronização de dados
Mantenha seus saves atualizados entre diferentes pastas ou máquinas.
📁 Gerenciamento organizado
Visualize, renomeie e organize facilmente seus perfis de jogo.
⚡ Leve e rápido
Aplicação simples, direta e eficiente para Windows.
⚠️ Aviso

O programa atua exclusivamente sobre arquivos locais de save. A “clonagem” refere-se à duplicação de dados no seu próprio computador, sem qualquer interação com servidores online.
